# Progressive Disclosure

## Progressive Disclosure

The end consumer is non-technical, but progressive disclosure allows digging into more technical settings. Settings use a three-tier progressive disclosure model: Everyday / Advanced / Expert. Protocol vocabulary (Nostr, relay, pubkey, WoT, npub, nsec) must never be exposed to non-technical users; 'Recovery Key' is used for nsec and 'Account address' for npub.
