# Image Filters

## Image Filters

Image filters are available when publishing posts. The app's 12 image filters are named Daylight, Ember, Dusk, Mist, Chrome, Film, Fade, Grain, Arctic, Copper, Veil, Bloom. Photo filters must be real CIFilter chains on iOS and real ColorFilter color matrices on Android — no stubs or placeholder implementations. Media encoding settings are JPEG quality 92, max dimension 2048px, with EXIF stripping. The default media server for publishing is blossom.primal.net, and media servers are configurable from the app settings.
